## 💡 如何获取正确的文件路径？

这是最关键的一步！有2种方法：

### 方法A：在「文件」App中查看（推荐）

```
① 打开「文件」App
② 找到 cat-video-guard 文件夹
③ 长按 index.html 文件
④ 选择「拷贝」
⑤ 打开「备忘录」App
⑥ 粘贴（Cmd+V 或 长按粘贴）
⑦ 你会看到类似这样的路径：

/var/mobile/Library/Mobile Documents/com~apple~CloudDocs/cat-video-guard/index.html

⑧ 复制这个路径！
⑨ 在前面加上 file:// 变成：

file:///var/mobile/Library/Mobile Documents/com~apple~CloudDocs/cat-video-guard/index.html?url=
```

### 方法B：使用更简单的方式（变通方案）

如果觉得路径太复杂，可以用这个**超级简化版**：

#### 超级简化版快捷指令（只需3步）：

```
快捷指令名称：🐱猫猫视频

动作1：「显示选取项目」对话框
  → 提示文字：「请选择cat-video-guard文件夹中的index.html」

动作2：「打开URL」
  → URL：点击魔法棒图标 → 选择「选取的项目」
  
动作3：（可选）「等待」1秒
```

**使用方式**：
1. 运行快捷指令
2. 弹出文件选择器
3. 导航到 cat-video-guard → 选择 index.html
4. 自动在Safari中打开！

## ❓ 常见问题

### Q: 提示"找不到文件"怎么办？

A: 检查路径是否正确：
- 确认 `cat-video-guard` 文件夹确实存在
- 确认 `index.html` 在该文件夹内
- 路径中的空格要用 `%20` 代替（如果有空格的话）

### Q: 可以添加到主屏幕吗？

A: 可以！
- 在快捷指令详情页，点击右上角 ⋯
- 选择「添加到主屏幕」
- 这样就像普通App一样了！

### Q: 想修改默认密码？

A: 
- 打开 `js/unlock-pin.js` 文件（在iPad的「文件」App中）
- 用文本编辑器找到 `setDefaultPIN(pin = '2024')`
- 改成你想要的密码
- 保存即可
