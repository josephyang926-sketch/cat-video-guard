# 🐱 喵视频守护者 - 使用指南

## 快速开始（3步搞定）

### 第1步：安装文件到iPad

**方法A：通过AirDrop/文件App**
1. 将整个 `cat-video-guard` 文件夹传到iPad
2. 使用「文件」App保存到「我的iPad」目录
3. 记住保存路径，例如：`/var/mobile/Library/Mobile Documents/com~apple~CloudDocs/cat-video-guard/`

**方法B：通过本地服务器（推荐开发者）**
```bash
# 在Mac上进入项目目录
cd cat-video-guard

# 启动本地服务器
python3 -m http.server 8080
```
然后在iPad Safari访问：`http://你的Mac的IP地址:8080`

---

### 第2步：添加到主屏幕（PWA模式）

1. 用Safari打开 `index.html`
2. 点击底部的 **「分享」按钮** (□↑)
3. 选择 **「添加到主屏幕」**
4. 名称改为：`🐱猫猫视频`
5. 点击 **「添加」**

✅ 现在桌面上会有一个像原生App一样的图标！

---

### 第3步：设置快捷指令（可选但推荐）

1. 将 `cat-video-guard.shortcut` 文件发送到iPad（AirDrop/邮件）
2. 双击打开 → 自动导入到「快捷指令」App
3. 打开快捷指令App，找到「🐱猫猫视频守护者」

---

## 🔐 解锁方式（双重保护）

### 第一层：四角手势解锁

当屏幕锁定时，你会看到：

```
┌─────────────────────────┐
│                         │
│    [1]           [2]    │  ← 按顺序点击四个角落
│                         │
│      🔒 屏幕已锁定       │
│                         │
│    [4]           [3]    │
│                         │
└─────────────────────────┘

点击顺序：左上(1) → 右上(2) → 右下(3) → 左下(4)
必须在3秒内完成
```

### 第二层：PIN密码输入

手势完成后，会弹出数字键盘：

```
┌──────────────────────┐
│     输入密码          │
│                      │
│     ● ● ● ●         │  ← 输入4位数字
│                      │
│   1 2 3             │
│   4 5 6             │
│   7 8 9             │
│     0 ⌫             │
└──────────────────────┘

默认密码：2024
```

### 完整解锁流程

```
视频锁定状态
    ↓
猫咪疯狂点击 😼
    ↓ (所有触摸被拦截)
屏幕显示：🔒 已锁定 + 四角提示
    ↓
你想操作时...
    ↓
步骤1: 依次点击 ↖↗↘↙ 四角
    ↓ (成功，震动反馈)
弹出PIN键盘
    ↓
步骤2: 输入密码 2024
    ↓ (正确)
✅ 解锁成功！可正常操作30秒
    ↓
30秒无操作后自动重新锁定
```

## ⚙️ 设置选项

解锁后可以调整设置（或点击界面中的设置入口）：

| 选项 | 说明 | 默认值 |
|------|------|--------|
| **屏蔽底部手势栏** | 防止猫滑出视频回到主屏 | ✅ 开启 |
| **屏蔽侧边滑动** | 防止左右滑动切换页面 | ✅ 开启 |
| **猫爪检测模式** | 自动识别猫爪行为并忽略 | ❌ 关闭 |
| **自动重新锁定** | 解锁后多久自动锁定（秒） | 30秒 |
| **修改PIN密码** | 自定义4-8位数字密码 | 2024 |

## 常见问题

### Q: 为什么有些网站无法加载？

**A**: 这通常是因为网站的 `X-Frame-Options` 或 `Content-Security-Policy` 安全策略禁止被嵌入到iframe中。

**解决方案**:
- 大多数主流视频网站（B站、YouTube等）是支持的
- 如果遇到不支持的网站，可以反馈给我来寻找替代方案

### Q: 如何修改默认密码？

**A**: 
1. 解锁屏幕
2. 进入设置面板
3. 点击「修改PIN密码」
4. 输入新的4-8位数字密码

或者：在PIN输入界面点击「忘记密码?」可重置为默认密码 `2024`

### Q: 猫太聪明了，学会了怎么办？😼

**A**: 
1. 开启「猫爪检测模式」- 会自动识别快速乱击并忽略
2. 可以考虑更换更复杂的手势序列（需要修改代码）
3. 终极方案：定制物理防护罩（亚克力板）

### Q: 会影响视频画质吗？

**A**: **完全不会！** 
- 保护层是100%透明的
- 不覆盖任何像素
- 只是拦截触摸事件
- 视频体验与正常观看完全一致

### Q: 支持横屏吗？

**A**: 
✅ 完全支持！
- 自动适配屏幕旋转
- 手势解锁区域会自动调整位置
- 键盘也会相应优化

## 开发者信息

如果你想要自定义功能或修改代码：

### 修改手势解锁序列
编辑 `js/unlock-gesture.js`:
```javascript
this.requiredSequence = ['tl', 'tr', 'br', 'bl']; // 当前：顺时针
// 可改为：['tl', 'bl', 'br', 'tr'] (Z字形)
// 或：['tr', 'tl', 'bl', 'br'] (逆时针)
```

### 修改默认PIN码
编辑 `js/unlock-pin.js`:
```javascript
setDefaultPIN(pin = '你的密码'); // 修改这里
```

### 调整屏蔽区域大小
编辑 `css/style.css`:
```css
.zone-bottom {
  height: max(env(safe-area-inset-bottom), 50px); /* 调整底部屏蔽高度 */
}
```
